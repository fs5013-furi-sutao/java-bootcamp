# 27. Java の配列

<p class="goal">配列のデータを使った処理を実装できるようになること</p>

## 学習

### 配列を知る

[01.Java の変数](./01-variable)で学習したように値を 1 つ格納するには変数を使います。
しかし複数の値を格納したい場合は、変数だけでは不便な場面も多いです。例えば次のような例を見てみましょう。

```java
String name1 = "山田";
String name2 = "鈴木";
String name3 = "佐藤";

int age1 = 20;
int age2 = 25;
int age3 = 30;

System.out.printf("%s さんは %d 歳です.\n", name1, age1);
System.out.printf("%s さんは %d 歳です.\n", name2, age2);
System.out.printf("%s さんは %d 歳です.\n", name3, age3);
```

### 実行結果

```
山田 さんは 20 歳です.
鈴木 さんは 25 歳です.
佐藤 さんは 30 歳です.
```

このプログラムでは 3 人の名前と年齢を変数に入れてから表示をしています。
ほぼ同じ名前の変数が複数存在しているので、少し管理が大変ですが無理ではないです。

では 100人、1000 人の情報を保持したい場合も同じ方法を使ったらどうでしょうか。
変数の管理が大変になり、記述しなければいけないプログラムの量も膨大になるのが想像できるはずです。
このような問題を解決してくれるのが「配列」です。

変数が「箱」に例えられるのに対して、配列は温泉施設などの「ロッカー」に例えられます。
温泉施設のロッカーには荷物を入れる場所が複数存在し、一つ一つに番号が割り振られています。
配列も同じようなイメージを持つと理解がしやすいです。

## 配列の宣言

配列の宣言は変数の宣言と似ていますが、データ型の後ろに角括弧を付ける必要があります。

```java
String[] names;
```

次に、配列に持たせる箱の数を決める必要があります。この数のことを「要素数」と呼びます。


つまり 3 つの値を入れたい場合は次のように要素が 3 つの配列であることを明記しなければいけないです。


```java
String[] names;
names = new String[3];
```

また、この 2 行のプログラムを省略して 1 行で書くことも出来ます。省略した形は次のようになります。

```java
String[] names = new String[3];
```

## 配列に値を代入

配列に値を代入する場合はどの要素に入れるかを記述します。要素を識別するために 0 から始まる連番を使います。これの数値のことを「添え字（そえじ）」と呼びます。

今回は 3 つの要素を持つ配列を作りましたので、添え字は 0 ~ 2 です。

```java
String[] names = new String[3];

names[0] = "山田";
names[1] = "鈴木";
names[2] = "佐藤";
```

## 配列の初期化

配列は変数と同じように「初期化」をすることもできます。初期化処理は次のように書きます。

```java
String[] names = {"山田", "鈴木", "佐藤"};
```

## 配列の値を出力

配列の値を出力する場合は、代入するときと同じように添え字を指定する必要があります。
全ての名前を表示する場合は次のように書きます。 

```java
String[] names = {"山田", "鈴木", "佐藤"};

System.out.println(names[0]);
System.out.println(names[1]);
System.out.println(names[2]);
```

### 実行結果

```
山田
鈴木
佐藤
```

## まとめ

最初に提示したプログラムは配列を使って書き直すと次のようになります。

```java
String[] names = {"山田", "鈴木", "佐藤"};
int[] ages = {20, 25, 30};

System.out.printf("%s さんは %d 歳です.\n", names[0], ages[0]);
System.out.printf("%s さんは %d 歳です.\n", names[1], ages[1]);
System.out.printf("%s さんは %d 歳です.\n", names[2], ages[2]);
```

### 実行結果

```
山田 さんは 20 歳です.
鈴木 さんは 25 歳です.
佐藤 さんは 30 歳です.
高橋 さんは 28 歳です.
山本 さんは 34 歳です.
```

### プラスα

配列はwhile文やfor文などの繰り返し処理と非常に相性が良いです。
for文などの使い方は後に出てきますが、ここで知っておいて欲しいのは配列とfor文などの繰り返し処理を組みわせると楽ができる点です。

下の例ではfor文を使うことでデータの数に関わらず、同じプログラムで配列の値を表示することができます。

```java
String[] names = {"山田", "鈴木", "佐藤", "高橋", "山本"};
int[] ages = {20, 25, 30, 28, 34};

for(int i = 0;i < names.length;i++) {
    System.out.printf("%s さんは %d 歳です.\n", names[i], ages[i]);
}
```
