## byte型を使ってみよう

以下はbyte型がとりうる最小値と最大値を表示するプログラムです。

- byte 型の変数「minByte」を byte 型のとりうる最小値で初期化
- byte 型の変数「maxByte」を byte 型のとりうる最大値で初期化

- 変数 `minByte` の内容を表示
- 変数 `maxByte` の内容を表示

```java
public class ByteType {
    public static void main(String[] args) {

        byte minByte = Byte.MIN_VALUE;
        byte maxByte = Byte.MAX_VALUE;

        System.out.printf("byte 型の最小値 = %4d %n", minByte);
        System.out.printf("byte 型の最大値 = %4d %n", maxByte);
    }
}

```

### プログラム実行結果

```
byte 型の最小値 = -128
byte 型の最大値 =  127
```
