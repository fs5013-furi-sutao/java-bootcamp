## 文字列の分割

splitメソッドを使うと文字列を分割することができます。この例では半角スペースで文字列を分割しています。

- 変数 「fullName」 を「山本 一郎」で初期化
- 変数 「fullName」 に 文字列を分割するメソッドを使用。半角スペースを引数にして呼び出し、分割結果を配列「names」に代入
- 配列「names」を使って、姓と名を表示

```java
public class StringSplitter {
    public static void main(String[] args) {

        String fullName = "山本 一郎";
        String[] names = fullName.split(" ");
        System.out.printf("姓: %s, 名: %s %n", names[0], names[1]);
    }
}
```

### StringSplitter の実行結果

```
姓: 山本, 名: 一郎
```
