# 53. 作成課題 ⭐ ハングマン

<p class="game-content">コンピュータがランダムで選んだ英単語を当てるゲーム</p>

<strong style="color: #ff0000;">
  この課題は作成が完了したら、教育担当者にコードレビューを依頼してください
</strong>

## 内容

コンピュータがランダムで選んだ英単語を当てるゲームです。

ゲーム開始時には単語の全ての文字が「*」として表示され、単語の文字数だけが分かります。

プレイヤーは、単語に入っていると思われるアルファベットを1つずつ答えます。

答えたアルファベットが単語に含まれる場合は、単語のどの位置にそのアルファベットが入っているのかが分かります。

単語に含まれる全てのアルファベットを答えることができればゲームクリアです。

※入力はアルファベット大文字で行うものとする。

※出題する英単語は次の配列の中からランダムで決めるものとする。

```java
String[] words = {
    "APPLE","BOOK","FRIEND","GARDEN","HOUSE",
    "FLOWER","JUMP","LION","MOUSE","ORANGE",
    "PENCIL","QUEEN","RABBIT","SCHOOL","TREE",
    "OCEAN","WATCH","GUITAR","YELLOW","BALL",
    "DESK","FISH","GOAT","LAMP","MONKEY",
    "NOSE","OCTOPUS","SNAKE","TABLE","VIOLIN"
};
```

### 実行例

```
英単語を当てよう！
******
文字を入力：E

*E****
文字を入力：A

*E****
文字を入力：O

*E**O*
文字を入力：L

*ELLO*
文字を入力：Y

YELLO*
文字を入力：W

正解！答えは YELLOW でした。
```

## チャレンジ

以下の改造を行ってみよう。

1. アルファベットを答えれるのは10回までに制限する。また、残りの回答回数は毎回表示すること。
2. アルファベット小文字でも入力できるようにする。

### データの管理方法

1.ランダムで選んだ英単語をchar型の配列に入れる。

2.単語の各文字が当てられたかを管理するためにはboolean型の配列を使う。初期値はfalseとして、文字を当てた場合はtrueに変更する。

※要素数は1.で作成したchar型の配列と同じにすること。
