# 09. Stream API

<p class="goal">
  Stream API を使って、中間処理、終端処理が実装できるようになること
</p>

## 学習

### Java 8 から導入されている Stream API を知る

Java における Stream API とは何かを Web で調べてください。
検索キーワードは、「 [java stream api](https://www.google.com/search?q=java+stream+api) 」です。
3 種類ぐらい説明記事やサンプルコードを読んで、Java の Stream API とは何か、 Stream API の使い方をつかみましょう。

加工したいデータを Stream化 すること、データを加工する(中間操作)、結果を得る(終端操作)ことを押さえておいてください。
