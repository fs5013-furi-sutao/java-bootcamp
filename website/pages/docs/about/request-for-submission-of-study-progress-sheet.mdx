# 学習進捗 Excel 表の提出のお願い

学習進捗 Excel 表の提出をお願いします

## お願い

<strong>⇒ <span style="color: red;">手元で管理している学習進捗 Excel 表を橋本の LINE に お送りください</span></strong>
<br />
（６月３０日の状態を想定して、完了日を埋めた状態でお願いします）

> 不明点などあれば、LINE メッセージや LINE 電話などで、何でも気軽に質問してください

<br />

## 提出期限

6/20(日) 23:59 まで

<br />

## 提出してもらう理由

7月に行う査定面談の場で使うためです。

<br />

## 確認が必要なシート
以下の4つのシートで完了している項目には、「完了日」欄にすべて完了日を入力しておいてください。

> - Java カリキュラム
> - Android アプリ
> - 基本情報
> - Web アプリ

<br />

## Excel アプリを持っていない場合は

Excel アプリを持っていなくて資料を編集できないよ、という方は、Excel Online を使ってください（Microsoft アカウントが必要です）。

参考: Excel Online （エクセル オンライン ）のサインイン方法  
https://www.9-soft.com/excel-online
