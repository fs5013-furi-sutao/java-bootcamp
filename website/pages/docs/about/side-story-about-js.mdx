# Java Bootcamp サイト作成こぼれ話

Java カリキュラムサイトのリニューアルについてのこぼれ話です

## 補足：JavaScript を学びたい方へ

Web ブラウザでページの見た目をぐいぐい動かす、JavaScript が好きな人に向けてのお話です。

> 注: Java カリキュラムについての話ではありません

## React によるユーザ体験の向上

このサイトのサイドバーメニューをポチポチクリックして、いろんな画面を遷移してみてください。

どうでしょうか？

あなたが普段使うサイトに比べて、遷移スピードが 💥 爆速 💥 だと思います。

このサイトでは、この遷移スピードの実現を次の 2 つの技術で行っています。

- React: Web ページ構築とページの使いやすさを実現する JavaScript フレームワーク
- GraphQL: データの取得更新を問い合わせる仕組み
