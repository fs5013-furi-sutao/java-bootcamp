# よくある質問

Java 学習カリキュラムを進める上でよくある質問をまとめています

## コードレビューはどう依頼すればいいの？

[Chatwork](https://go.chatwork.com/ja/) から教育担当者あてにレビュー依頼をしてください。
さらに、LINE から教育担当者に依頼しましたよ、のメッセージを投げかければ、早くリアクションができると思います。

LINE のみでは、過去の質問記録が残らないので、必ず [Chatwork](https://go.chatwork.com/ja/) のほうに依頼メッセージを残してください。

## サンプルコードや解答例で分からないことがあるんだけど

分からないことがあれば随時、聞いてください。
コードレビューと同様に [Chatwork](https://go.chatwork.com/ja/) や LINE でメッセージをください。

## Chatwork はどうやって始めたらいいの？

[Chatwork の導入手順](./how-to-start-chatwork) を参考にアカウント登録、開始をしてください。
