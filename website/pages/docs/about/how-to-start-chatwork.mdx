# Chatwork の導入手順

社内チャットツールとして採用しているチャットワークの始め方を説明します

## アカウント作成方法

### ① ブラウザで Chatwork のログインページを開く

ログインページ  
https://www.chatwork.com/login.php

### ② 新規登録（無料）ボタンをクリック

### ③ メールアドレスを入力

freestyles.jp の自分のメールアドレスを入力して「次へ進む」をクリック

新規登録の流れを説明するページに遷移したことを確認する

この時点で、登録したメールアドレスにメールが届く

### ⑤ 届いたメールの「アカウント登録」ボタンをクリック

本登録を行うページが開くので、必要事項を記入する

### ⑥ 「同意して始める」ボタンをクリック

### ⑦ 登録完了メールが届いたのを確認する

### ⑧ ログインできることを確認する

ログインページ  
https://www.chatwork.com/login.php

### ⑨ 画面右上のコンタクト申請ボタンをクリック

右上ヘッダ部にコンタクト申請ボタン（人の形に＋マークが付いたもの）をクリックする

### ⑩ コンタクト申請内容を入力

メールアドレスとメッセージを入力するフォームが現れる

以下のメールアドレスを入力
technical_education_support@freestyles.jp

以下のメッセージを入力
「学習カリキュラムを始めたいので、コンタクト承認をお願いします」

### ⑪ コンタクト承認されるまで待つ

コンタクト承認をして、教育担当者からグループチャットへ案内します

## その他

Chatwork にはスマホ向けのアプリもあります。
PC よりスマホをよくチェックする人は、スマホアプリを導入してください。
