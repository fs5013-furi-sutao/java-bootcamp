# Java Bootcamp へようこそ

Java Bootcamp は、およそ 250 時間で Java 言語とプログラミングの 基礎 ～ 発展的内容までの知識を手に入れる、フリースタイル ITS 所属の方向けのプログラムです。

## 🥇 学習の目標

Java Bootcamp での勉強は、以下の実力を養成することを目的としています。

- プログラミングの考え方をできるようになること
- オブジェクト指向を使いこなせるようになること
- Java 言語でプログラム設計ができること
- 複数のクラスを使った設計／実装ができるようになること
- 読みやすく、改修しやすいプログラムが書けるようになること

具体的には、🔥 Android アプリ制作や、Java 以外の言語も使える 🔥 水準の実力を付けることが目標です。

## 🐹 学習の流れ

サイドバーメニューのスタートから順番に学習を順番に進めていってください。

### 練習問題について
各項目には、練習問題があります。各項目を理解しているか、確認するためと、小さなコードをたくさん書くことで、コーディング作業を手に覚えさせることが、練習問題の目的です。** 練習問題の解答例 ** は、各ページに記載してあるリンク先で確認することができます。

### 課題について
各マイルストーンごとに、区切りとして、ゲームなどを作る課題も用意しています。この課題に関しては、解答を用意していません。書く課題が完成した際には、教育担当者にレビュー依頼をしてください。

### 課題のコードレビューについて
コードレビューでは、学習で基礎を習得し、品質の高いコーディングができているかをチェックします。このコードレビューで指摘された事項を修正していくことで、どんどんスキルアップをしていくことができます。

### 質問・コードレビュー歓迎

分からないことがあれば、いつでも気軽に質問してください。
また、課題についても、「すべてが完成していないけど一度、これでいいのか見てほしい」から、レビュー依頼をするというのも大歓迎です。

間違った方向で細部まで作り込んでしまってから、コードレビューで間違いを指摘され修正するのは、とても効率が悪いです。
作り込んだものを修正しようとすると、変更する時の影響が他に及びやすく最悪、身動きが取れなくなります。

コードレビューや質問で、間違いは早めに軌道修正していくのが、効率の良い学び方です。

## 🐹 Excel で進捗管理をしてください

どの学習項目がいつ完了したのか、後で分かるように、完了した日付を記録しておいてください。

過去どこの時点で、どこの項目まで終わっていたのか、学習記録として重要になるので、必ず各学習項目の完了日を記録ください。

完了日の記録は以下の、学習進捗 Excel 表を利用してください。

[学習進捗記録_サンプル.yyyy.mm.dd.xlsx](./学習進捗記録_サンプル.yyyy.mm.dd.xlsx)

Excel が使えない場合は、以下のテキストファイルに完了日を記録しておいてください。

[titles-and-finished-date.txt](./titles-and-finished-date.txt)

### 2023年3月以前の記録について
「Javaカリキュラム」「Androidアプリ」「Webアプリ」シートそれぞれの「完了日」列をコピーして、
以下からダウンロードできる新しい進捗記録の各シートに貼り付けてください。

### 学習進捗 Excel 表への完了日の記載方法：Javaカリキュラム,Androidアプリ,Webアプリ

Excel ファイルの「Java カリキュラム」シートの「完了日」列（黄色背景のセルです）に日付（yyyy/mm/dd 形式）を記載すると、「完了」列に「●」が表示され、
「消化」列に消化した学習工数が表示されます。

消化した学習工数と実学習時間が異なる場合は、さらに隣の「実学習時間」列に実際の学習時間をご記入ください。

### 学習進捗 Excel 表への完了日の記載方法：自主学習項目

「自主学習サンプル」シートをコピーして、名前を自分の学習する項目に書き換えてください。

B列に学習した内容を簡潔に記載し、C列に日付、D列に所要時間を記載してください。

### 月ごとのメンテナンスの説明

この Excel シートには、月ごとの進捗のパーセンテージを自動集計する機能が付いています。

自動集計といっても、各自で以下のメンテナンスが必要です。

- 完了した学習項目への完了日付の記載（随時）
- 自主学習項目集計シートでの学習項目の追加、月ごとの行コピー・行挿入
- 進捗集計シートへの行コピー・行挿入（月末ごと）

### 毎月1日に学習報告

月に一度、学習がどれだけ進んでいるか、以下の方法で学習報告をしてください。

学習報告は、あなたの学習を後押しするための仕組みです。

- 学習進捗記録の Excel をメールに添付して送付（毎月1日）

Excel ファイルを送付する時は、ファイル名を自分の名前、送付する年月にしておいてください。

たとえば、山本一郎さんが 2022 年 2 月分の進捗を集計して、3 月 1 日に進捗報告を送付する場合は、次のようなファイル名です。

```
学習進捗記録_山本一郎.2022.02.28xlsx
```
