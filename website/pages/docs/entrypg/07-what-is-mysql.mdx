# 07. MySQLとは

<p className="goal">MySQLとは何かを知る</p>

## 学習

### MySQLとは

MySQLは、データの管理と格納に広く使用されているオープンソースのリレーショナルデータベース管理システム（RDBMS）です。
名前こと難しいですが、データを効率良く管理するためのシステムです。

普段私たちが使っている様々なサービスにRDBMSが広く使われています。
RDBMSにはOracle Database、SQL Server、PostgreSQLなど様々なものが存在していますが本試験ではMySQLを使います。

MySQLを使うためにDBサーバ（データベースサーバ）を準備する必要があります。
ここでは軽量で高速に動作するコンテナ型仮想環境用のプラットフォームであるDockerを使用します。

Dockerについて詳しく知りたい方はWebで調べてください。検索ワードは、「 [Dockerとは](https://www.google.com/search?q=Dockerとは) 」です。

また、データベースの学習を進めるうえで、一般的に使用する用語も確認しておきましょう。検索ワードは、「 [データベース　用語](https://www.google.com/search?q=データベース+用語) 」です。

ここで最初に押さえておきたい用語は「テーブル」「カラム」「レコード」「フィールド」です。
