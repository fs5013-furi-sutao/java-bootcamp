# 11. SELECT文の書き方
データベースから値を取得する方法を説明する

## SQL文を実行する方法

A5:SQL Mk-2を使うとSQL文を実行することができます。

実行するためには、画面左上の`ファイル`をクリックします。

ドロップダウンメニューの中の`新規`をクリックします。

ポップアップウィンドウの`SQL`をクリックします。

SQL文の入力欄が表示されますので、実行したいSQL文を入力し、入力欄左上の`SQLを実行`ボタンをクリックします。正しく入力ができていれば、実行結果が表示されます。

![SQL文の実行](./execute-sql-command.png)

## 基本的な書き方

テーブルから値を取得する場合は、SELECT文を使います。まずはSELECT文の一番簡単な書き方を確認しましょう。

指定したテーブルから指定したカラム（列）だけを取得する場合は次のように書きます。

```sql
SELECT カラム名
FROM テーブル名;
```

### 1.テーブルから一つのカラムを取得する
`employees`テーブルから`firstName`カラムの値を取得する場合は次のように書きます。
ここでは`employees`テーブルから`firstName`が23件（全件）取得し、表示しています。

```sql
SELECT firstName
FROM employees;
```

### 2.テーブルから複数をカラム取得する
データを取得する場合は、同時に複数のカラムを取得することもできます。
`employees`テーブルから`firstName`と`lastName`カラムの値を取得する場合はカラム名をコンマ区切りで記述します。

```sql
SELECT firstName, lastName
FROM employees;
```

### 3.テーブルからすべてのカラムを取得する
テーブルからすべてのカラムを取得したい場合は、カラム名の代わりに`＊`(アスタリスク)を記述します。

```sql
SELECT *
FROM employees;
```

### 4.条件を一つ指定してレコードを取得する
ここまでのSQL文はすべてのレコード（行）を取得する内容でしたが、条件を追加して、条件に当てはまるレコードだけを取得することができます。
例えば、`employees`テーブルから`officeCode`が`2002`であるレコードだけを取り出す場合は次のように書きます。

```sql
SELECT *
FROM employees
WHERE officeCode = '2002';
```

※文字列と比較する場合に文字列を`'`シングルクォートで囲む必要があります。

また、比較をするための演算子がいくつか存在しますので、詳しく知りたい方はWebで調べてください。検索ワードは、「 [MySQL 演算子](https://www.google.com/search?q=MySQL+演算子) 」です。

### 5.条件を複数指定してレコードを取得する
複数の条件を指定して検索することもできます。その場合はANDやORを使って、条件を繋げることができます。

`officeCode`が`2001`かつ、`reportsTo`が`1143`のレコードを取り出す。
```sql
SELECT *
FROM employees
WHERE officeCode = '2001'
  AND reportsTo = 1143;
```

`officeCode`が`2001`または、`reportsTo`が`1143`のレコードを取り出す。
```sql
SELECT *
FROM employees
WHERE officeCode = '2001'
   OR reportsTo = 1143;
```

### 6.主キーを指定してレコードを取得する

レコードを取得する場合は、条件の書き方によって、同時に複数のレコードを取得することができます。

しかし特定のレコードを1件だけ取得したい場合もあります。そのような場合に使うのが主キー（プライマリーキーとも言う）です。

主キーとは重複した値を登録することができない、特別な制約が設けられたカラムです。

このような制約が設けられているため主キーを指定してレコードを取得すれば、該当する1件のレコードだけを取り出すことができます。

![主キーについて](./show-table.png)

A5 SQL:Mk-2では主キーとなっている項目は背景色が赤色で表示されます。

また、`制約`タブを開くとテーブルに設定されている制約の一覧が確認でき、主キーには`PRIMARY KEY`という種類の制約が設定されている。
