# 01. 単体テストとは

<p class="goal">
  単体テストのテストケースの作成方法を知る
</p>

## 学習

### テストとは

テストとは作成したソフトウェアが正しく作られているかを確認する工程のことです。
開発手法によってテストの仕方も少し変わりまずが、本試験ではV字モデルを使った開発を想定しています。
詳しくはWebで調べてください。検索ワードは、「 [システム開発　V字モデル](https://www.google.com/search?q=システム開発+V字モデル) 」です。

### 単体テストとは

上の項目で確認できたように、テストにも単体テスト・結合テスト・システムテストなど複数の工程があります。

本試験では作成した機能の単体テストを行います。
単体テスト（ユニットテストともいう）はプログラムを構成する最小単位である「モジュール」が正しく作られているかを確認するものです。
Javaの場合、一般的にメソッド単位またはクラス単位のテストを指します。
