## 学習

### ⭐ を知る

⭐ とは何か、Java で ⭐ を使う方法を Web で調べてください。検索キーワードは、「 [java 配列](https://www.google.com/search?q=java+配列) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、⭐ とは何かを知り、⭐ の使い方をつかみましょう。
