# 13. コンパイル方法

<p class="goal">
  コンパイルとは何か、コンパイルの方法が分かり、コンパイルを実施できるようになること
</p>

## 学習

### コンパイル方法を学ぶ

コンパイルとは何かということと、コンパイルの方法を Web で調べてください。検索キーワードは、「 [java コンパイル](https://www.google.com/search?q=java+コンパイル) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、コンパイルするとはどういうことか、コンパイルはどうやってやるのかを習得しましょう。

## 練習問題: コンパイルを実践する（時間: 20 分）

以下のプログラム TargetOfCompile.java について、コマンドライン（コマンドプロンプト、PowerShell、GitBash など）でコンパイルをして、クラスファイルを実行しなさい

### TargetOfCompile.java

```java
public class TargetOfCompile {
    public static void main(String[] args) {

        System.out.println("コンパイルの動作テストです.");
    }
}
```

### コンパイル後の実行結果

```console
コンパイルの動作テストです.
```
