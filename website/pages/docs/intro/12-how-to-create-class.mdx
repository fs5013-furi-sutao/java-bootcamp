# 12. クラスの作成方法

<p className="goal">VSCode を使って、クラスの作成ができること</p>

## 学習

直前の学習項目で作成したプロジェクト内に新しいクラスを作成します。

### 1. 新しいファイルの作成

サイドバーの `src` フォルダを右クリックし、
ポップアップメニューの `新しいファイル...` をクリックする。

![新しいファイルの作成](./create-class.png)

### 2. ファイル名の入力

ファイル名の入力を求められるので、「FirstClass.java」と入力し、 Enter を押す。

この場合は「FirstClass」はクラスの名前で「.java」は拡張子です。

### 3. 作成したクラスファイルの内容を確認

クラスの作成が終わると次のように表示される。

このプログラムはVSCodeにより自動入力されるが、自動入力されない場合は手動で入力する。

このとき「`public class 〇〇`」の〇〇の部分をファイル名と同じにする必要がある。

```java
public class FirstClass {
    
}
```

### 4. メインメソッドの入力

次にメインメソッドを入力する。

メインメソッドはプログラムの実行が始まる部分だと覚えておこう。

現時点では意味について深く考えずに「必ず必要になるおまじない」という認識で問題はない。

```java
public class FirstClass {
    public static void main(String[] args) {

    }
}
```

### 5. 実行したいプログラムを入力

最後にメインメソッドの中に実行したいプログラムを入力する。

今回は「`Hello, Free Style!`」と表示するために、次のプログラムを入力しよう。

```java
public class FirstClass {
    public static void main(String[] args) {
        System.out.println("Hello, Free Style!");
    }
}
```

### 6. プログラムを実行

プログラムを実行し、「`Hello, Free Style!`」と表示されることを確認したら完成です。

このように新しいプログラムを入力する場合は、新しいクラスを作成し、その中にプログラムを入力する。
