# 16. 作成課題 ⭐ ブラックジャック

<p className="game-content">コンピュータとのブラックジャック勝負</p>

<strong strong style={{color: "#ff0000"}}>
  この課題は作成が完了したら、教育担当者にコードレビューを依頼してください
</strong>

## 内容

コンピュータとのブラックジャック勝負です。

プレイヤーとコンピュータそれぞれに、カードが２枚配られます。

「A」は「1 または 11」として計算でき、「J ～ K」は「10」として計算します。

カードの合計が「21」に近い方が勝ちです。

プレイヤーは、カードの合計が「21」未満の場合は、もう 1 枚引くか、引かないかを選ぶこ
とができます。

ただし、「21」を超えた場合は、バーストとなります。

ディーラーは１７以下の場合は必ずカードを引きます。

## 課題 1

トランプのカードを表すクラスを作成する。
ジョーカーは含めない。

| プロパティ | 型         | 備考                                                                  |
| :--------- | :--------- | :-------------------------------------------------------------------- |
| mark       | 列挙型     | ダイヤ(Diamonds)、ハート(Hearts)、クローバー(Clubs)、スペード(Spades) |
| number     | 数値(1-13) | 1 は A、11 は J、12 は Q、13 は K                                     |

※「列挙型」（別名：Enum）は　「 [中級編17. Enum クラス](../advanced/17-enum-class) 」 で詳しく学習します。
ここでは簡単に使い方を検索して、使ってみましょう。
検索キーワードは、「 [java enum](https://www.google.com/search?q=java+enum) 」です。

## 課題 2

トランプを管理するクラスを作成する。
カードの管理には、課題 1 で作成したカードクラスを使用する。

| メソッド | 説明                                       |
| :------- | :----------------------------------------- |
| reset    | トランプをリセットする（残り 52 枚にする） |
| shuffle  | 残りのトランプをシャッフルする             |
| draw     | カードを１枚引く                           |

## 課題 3

プレイヤーまたはディーラーが引いたカードの点数を求めるために次のメソッドを作成する。
カードのリストを引数として受け取り、点数を数えて返すメソッド。

カードの点数は以下のとおり。

| カード  | 点数                                         |
| :------ | :------------------------------------------- |
| A       | 1 点または 11 点(どちらか都合のいい方にする) |
| J、Q、K | 10 点                                        |
| 2 ～ 10 | 数字通りの点数                               |

課題 2 までで作成したトランプの管理クラスとカードクラスを使用する。

## 課題 4

課題 1 ～ 3 までで作成したクラスを使ってブラックジャックを完成させる。

### 実行例

```
あなたに「ハート 8」が配られました。
ディーラーに「ダイヤ A」が配られました。
あなたに「スペード 2」が配られました。
ディーラーに「ハート 5」が配られました。

ディーラーの合計は 15 です。
現在の合計は 10 です。
もう１枚カードを引きますか？(Y/N)：Y

あなたに「クローバー 9」が配られました。
現在の合計は 19 です。
もう１枚カードを引きますか？(Y/N)：N

ディーラーに「ダイヤ J」が配られました。
ディーラーの合計は 15 です。

ディーラーに「クローバー 3」が配られました。
ディーラーの合計は 18 です。

あなたの勝ちです
```
