## 学習開始にあたって用意するもの

この Java 学習カリキュラムを開始するにあたって、以下のものを準備しておいてください。

- PC（このサイトでの説明は Windows 10 Home を想定）
- Java 開発環境（カリキュラム内で手順を説明しています）
- Chatwork アカウント（教育担当者への質問やコードレビュー依頼に使用します）

Chatwork アカウントは、FreeStyle の個人アドレスで作成しておいてください。
アカウント登録方法、コンタクト申請方法は、別ページの [Chatwork の導入手順](./about/how-to-start-chatwork)を参考にしてください。
